# Amodal Scene Understanding for Depth-Ordered Tracking

This project focuses on **amodal scene understanding for depth-ordered multi-object tracking under occlusion**, addressing a key limitation in traditional computer vision pipelines: the inability to maintain object identity when objects become partially or fully hidden. Standard (modal) detection systems only perceive visible regions, which leads to tracking failures during occlusion events. To overcome this, the pipeline leverages amodal perception to infer the full extent of objects—even when they are not fully visible—enabling more robust reasoning about scene structure and object continuity over time.

The core of the system is a three-stage pipeline that integrates **amodal segmentation, depth estimation, and tracking**. For segmentation, **C2F-Seg** is used to generate high-quality amodal masks that reconstruct occluded object regions beyond what is directly visible. These masks are then passed to **DepthAnything**, a state-of-the-art monocular depth estimation model, which produces dense depth maps used to derive relative depth ordering between objects. By computing depth statistics within each amodal mask, the system establishes a layered representation of the scene, allowing it to reason about which objects are in front or behind others during occlusion events.

Finally, the pipeline uses **OC-SORT**, an occlusion-aware multi-object tracking algorithm, to maintain consistent object identities across frames. Unlike traditional trackers, OC-SORT is specifically designed to handle periods of missing observations caused by occlusion, making it well-suited for this task. By combining amodal masks, depth-based ordering, and robust tracking, the system demonstrates improved performance over standard modal baselines, particularly in scenarios involving prolonged or complex occlusions. This integrated approach highlights the practical value of amodal perception for dynamic scene understanding and tracking in real-world environments.
